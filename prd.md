# ANGI — Product Requirements Document

## Overview

ANGI is an e-commerce website for a Tamil streetwear brand that sells apparel and accessories. The brand is rooted in Tamil cultural identity with two main lines: **ANGI** (Tamil-focused) and **Identity** (minimal/lifestyle).

## Brand Philosophy

- **Core Values:** Authenticity, Identity, Culture
- **Tagline:** "You Are You"
- **Mission:** Establishing identity through garments, arts, exhibitions and music
- **Target Audience:** Young adults (18-35) in India interested in streetwear and Tamil cultural expression

## Product Categories

| Category | Description |
|---|---|
| Tamil T-Shirts | Bold Tamil typography, cultural artwork, graphic prints — the ANGI mainline |
| Identity T-Shirts | Minimal, clean designs from the Identity sub-brand |
| Hoodies | Premium heavyweight fleece with graphic prints and special finishes |
| Polo T-Shirts | Classic polo silhouettes with embroidered ANGI branding |
| Keychains | Metal accessories with ANGI logo and Tamil script |

## Print Techniques Used

- Digital Print with Silicon Softener
- Non-PVC & Puff Print
- HD Print
- Acid Wash (Lava-Dye) + Digital Print
- Discharge Print with Dye-Spray
- Pigment Print + Reverse CPD
- Flock & HD Sticker
- Embroidery

## Design Direction

- **Style:** Luxury minimal (inspired by Represent CLO, Zara, H&M)
- **Color scheme:** White background, black text (#111), grey hierarchy (#999/#bbb/#ccc)
- **Typography:** Small (9-12px), wide letter-spacing, uppercase labels, font-extralight for logo
- **Layout:** Edge-to-edge product grids, 1px gaps, full-bleed hero images, no visible card borders
- **Photography:** Model shots on neutral backgrounds

## Currency & Region

- **Primary currency:** INR (₹)
- **Region:** India
- **Free shipping threshold:** ₹1,999

## Tech Stack

- **Frontend:** Next.js 16 + Tailwind CSS + TypeScript
- **Backend (planned):** Zoho Commerce for product management, cart persistence, checkout, and payments
- **Hosting:** TBD (Vercel recommended for Next.js)
